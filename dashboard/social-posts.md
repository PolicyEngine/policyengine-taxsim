# Social posts: PolicyEngine TAXSIM landing page launch

## LinkedIn

### Option A (developer-focused)

We just launched a new site for PolicyEngine's open-source TAXSIM emulator: policyengine.org/us/taxsim

`pip install policyengine-taxsim` gives you a drop-in replacement for TAXSIM35. Same stdin/stdout interface, same input variables, same output format. Just swap the command name.

Works in CLI, Python, R, Stata, SAS, and Julia -- no code changes needed beyond the command name.

Built under a memorandum of understanding with NBER and TAXSIM creator Daniel Feenberg, and cross-validated with the Federal Reserve Bank of Atlanta's Policy Rules Database.

Fully open source: github.com/PolicyEngine/policyengine-taxsim

### Option B (research-focused)

PolicyEngine's TAXSIM emulator now has a home: policyengine.org/us/taxsim

For researchers who use TAXSIM to calculate federal and state income taxes, this is a drop-in replacement that runs the same inputs and outputs through PolicyEngine's microsimulation engine.

What's different:
- Fully open source -- inspect every calculation
- Covers programs beyond income taxes: SNAP, Medicaid, CHIP, ACA subsidies, EITC, CTC, and hundreds more
- Built under an MOU with NBER and cross-validated with the Atlanta Fed's Policy Rules Database

Install with `pip install policyengine-taxsim` and swap `taxsim35` for `policyengine-taxsim`. Works in CLI, Python, R, Stata, SAS, and Julia.
