# R Package for PolicyEngine TAXSIM

This directory contains the R wrapper package for the PolicyEngine TAXSIM emulator.

## Structure

```
r-package/
└── policyenginetaxsim/     # R package source
    ├── DESCRIPTION         # Package metadata
    ├── NAMESPACE           # Exports and imports
    ├── LICENSE             # MIT license
    ├── R/                  # R source files
    │   ├── setup.R         # setup_policyengine(), check_policyengine_setup()
    │   ├── calculate_taxes.R  # policyengine_calculate_taxes()
    │   └── zzz.R           # Package load hooks
    └── man/                # Documentation (generated by roxygen2)
```

## Installation

```r
# From GitHub
devtools::install_github("PolicyEngine/policyengine-taxsim", subdir = "r-package/policyenginetaxsim")

# Or from local clone
devtools::install("r-package/policyenginetaxsim")
```

## Development

To regenerate documentation after editing roxygen comments:

```r
devtools::document("r-package/policyenginetaxsim")
```

To run tests:

```r
devtools::test("r-package/policyenginetaxsim")
```

To check the package:

```r
devtools::check("r-package/policyenginetaxsim")
```
